### Webpack 5 study

#### how to start
```js
npm i
or
npm install // install npm packages
```

```js
npm run build // build local project
```

```js
npm run start // start local app
```



### Current tips
- Entry point: input
- Output: output
- Plugins: thiry party widgets
- Module: a place to coimple css/js/html code  
- environments: dev/prod
